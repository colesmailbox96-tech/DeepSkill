# DeepSkill (Veilmarch Prototype)

Initial implementation scaffold for the Veilmarch low-poly RPG project.

## Current Scope

- Vite + React + TypeScript project baseline
- Three.js rendering skeleton
- Minimal world scene with:
  - perspective camera,
  - ambient + directional lighting,
  - flat terrain test plane,
  - continuous render loop

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
